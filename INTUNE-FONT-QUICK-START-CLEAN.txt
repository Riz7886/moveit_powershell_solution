HOW TO USE (SUPER SIMPLE):
===========================

STEP 1: GET THE FONT FOLDER
----------------------------
Receive folder with font files
Save it somewhere
Example: C:\Fonts


STEP 2: RUN THE SCRIPT
-----------------------
Open PowerShell as Administrator
Run: .\Deploy-Fonts-Intune-Automated.ps1


STEP 3: SELECT OPTION FROM MENU
--------------------------------
You'll see:

  [1] Deploy Fonts to Windows Devices
  [2] Deploy Fonts to Mac Devices
  [3] Deploy Fonts to ALL Devices (Windows + Mac)  ← RECOMMENDED
  [4] Check Deployment Status
  [5] Exit

Type: 3
(This deploys to both Windows AND Mac)


STEP 4: SELECT SUBSCRIPTION
----------------------------
Script shows your subscriptions:

  [1] Production
  [2] Staging
  [3] QA

Type: 1 (for Production)


STEP 5: POINT TO FONT FOLDER
-----------------------------
Script asks: "Font folder path:"

Type: C:\Fonts (or wherever fonts are saved)
Press Enter


STEP 6: REVIEW AND CONFIRM
---------------------------
Script shows:
- How many fonts found
- How many devices will get them
- Summary of what will happen

Type: DEPLOY
Press Enter


STEP 7: DONE!
-------------
Script does everything automatically:
✓ Creates Intune package
✓ Uploads fonts
✓ Assigns to all devices
✓ Everyone gets fonts automatically!

Total time: 5-10 minutes
Fonts deploy to all devices: 24-48 hours

WHAT HAPPENS ON EACH PLATFORM:
===============================

WINDOWS:
--------
1. Device checks in with Intune
2. Downloads font package
3. Runs PowerShell script
4. Copies fonts to C:\Windows\Fonts\
5. Registers fonts in registry
6. Fonts available in all apps
7. Done!

MAC:
----
1. Device checks in with Intune
2. Downloads font package
3. Runs shell script
4. Copies fonts to /Library/Fonts/
5. Updates font cache
6. Fonts available in all apps
7. Done!

===========

5. Click on it

Shows:
- Total devices
- Installed count
- Pending count
- Failed count
- Percentage complete


TROUBLESHOOTING:
================

ISSUE: Device doesn't get fonts
SOLUTION:
1. Force sync device in Intune portal
2. Or on device: Settings > Accounts > Access work or school > Sync
3. Fonts install within minutes

ISSUE: Fonts not showing in apps
SOLUTION:
1. Close and reopen app (Word, Excel, etc.)
2. Fonts should appear
3. If not, restart device

ISSUE: Script error
SOLUTION:
1. Make sure you're Intune admin
2. Make sure devices enrolled in Intune
3. Check error message and retry
