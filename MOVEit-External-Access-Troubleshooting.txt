FIX #2: MANUAL VIA AZURE PORTAL
================================

STEP 1: Open Azure Portal
--------------------------
1. Go to portal.azure.com
2. Login with your account

STEP 2: Navigate to NSG
------------------------
1. Click "Resource groups"
2. Click "rg-moveit" (or your MOVEit resource group)
3. Click "nsg-moveit" (Network Security Group)

STEP 3: Fix Inbound Rules
--------------------------
1. Click "Inbound security rules" on left menu
2. Find rule "Allow-HTTPS-443"
3. Click on the rule
4. Change "Source" from whatever it is to: "Any" or "Internet"
5. Click "Save"

STEP 4: Fix SFTP Rule
----------------------
1. Find rule "Allow-SFTP-22"
2. Click on the rule
3. Change "Source" from whatever it is to: "Any" or "Internet"
4. Click "Save"

STEP 5: Verify Rules
--------------------
You should now see:
✓ Allow-SFTP-22: Port 22, Source: Any, Action: Allow
✓ Allow-HTTPS-443: Port 443, Source: Any, Action: Allow

STEP 6: Test Access
-------------------
1. Wait 1-2 minutes for changes to propagate
2. From home computer, open browser
3. Go to: https://moveit.pyxhealth.com
4. Should now load! ✅

DETAILED RULE CONFIGURATION:
============================

RULE 1: Allow-SFTP-22
---------------------
Name: Allow-SFTP-22
Priority: 100
Port: 22
Protocol: TCP
Source: Any (or 0.0.0.0/0 or Internet)
Destination: Any (or VirtualNetwork)
Action: Allow

RULE 2: Allow-HTTPS-443
-----------------------
Name: Allow-HTTPS-443
Priority: 110
Port: 443
Protocol: TCP
Source: Any (or 0.0.0.0/0 or Internet)
Destination: Any (or VirtualNetwork)
Action: Allow

WHAT WENT WRONG?
================

MY MISTAKE:
The deployment scripts I created likely set NSG source to:
- "VirtualNetwork" (internal only)
- Or specific IP ranges

Instead of:
- "Internet" (allows external access)
- "Any" (0.0.0.0/0)

WHY THIS HAPPENED:
NSG rules can be configured for:
1. Internal-only access (VirtualNetwork)
2. Specific IP allowlist
3. Public Internet access (Any/Internet)

The scripts likely defaulted to #1 or #2 for security.

BUT FOR MOVEIT TO WORK EXTERNALLY:
Users at home need to access MOVEit → Must allow Internet!
