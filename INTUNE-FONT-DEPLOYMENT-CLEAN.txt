HOW IT WORKS:
=============

STEP 1: AUTO-INSTALL MODULES
-----------------------------
Script automatically installs required PowerShell modules:
- Az.Accounts (Azure authentication)
- Az.Resources (Azure resource management)
- Microsoft.Graph.Authentication (Graph API auth)
- Microsoft.Graph.DeviceManagement (Intune access)
- Microsoft.Graph.Intune (Intune management)

No manual installation needed!

STEP 2: MAIN MENU
-----------------
Shows deployment options:
[1] Deploy Fonts to Windows Devices
[2] Deploy Fonts to Mac Devices
[3] Deploy Fonts to ALL Devices (Windows + Mac)
[4] Check Deployment Status
[5] Exit

STEP 3: SUBSCRIPTION SELECTION
-------------------------------
Shows all Azure subscriptions (like AVD/MOVEit):

Available subscriptions:
  [1] Production
  [2] Staging
  [3] QA

You select which subscription to use

STEP 4: CONNECT TO INTUNE
--------------------------
Automatically connects to Microsoft Graph API
Required permissions:
- DeviceManagementManagedDevices.ReadWrite.All
- DeviceManagementConfiguration.ReadWrite.All
- DeviceManagementApps.ReadWrite.All

STEP 5: SELECT FONT FOLDER
---------------------------
You specify where your font files are located
Example: C:\Fonts

Supported formats:
- TTF (TrueType Font)
- OTF (OpenType Font)
- WOFF (Web Open Font Format)
- WOFF2 (Web Open Font Format 2)

STEP 6: AUTO-DISCOVER DEVICES
------------------------------
Script automatically finds ALL Intune-managed devices
Filters by platform based on your menu choice:
- Option 1: Windows devices only
- Option 2: Mac devices only
- Option 3: All devices

Shows count and sample list

STEP 7: CREATE DEPLOYMENT PACKAGE
----------------------------------
Automatically creates:
- PowerShell install script
- Intune Win32 app package
- Installation commands
- Uninstall commands

STEP 8: UPLOAD TO INTUNE
-------------------------
Uploads font package to Intune
Creates application in Intune portal
Makes it available for deployment

STEP 9: ASSIGN TO DEVICES
--------------------------
Automatically assigns to all discovered devices
Deployment type: Required (automatic)
Devices will install fonts automatically

STEP 10: COMPLETION
-------------------
Shows summary
Provides monitoring instructions
Displays timeline

TROUBLESHOOTING:
================

ISSUE: No devices found
SOLUTION:
- Make sure devices are enrolled in Intune
- Check Intune > Devices > All devices
- Devices must be Azure AD joined or Intune MDM enrolled

ISSUE: Fonts not installing
SOLUTION:
- Check device sync status (Devices > Select device > Sync)
- Verify device OS version is supported
- Check installation logs on device
- Windows: Event Viewer > Applications and Services Logs > Microsoft > Windows > DeviceManagement-Enterprise-Diagnostics-Provider
- Mac: Console app > filter for "Intune"

ISSUE: Script fails to upload
SOLUTION:
- Check permissions (Intune Administrator role)
- Verify Graph API connection
- Re-run: Connect-MgGraph -Scopes "..." manually
- Check font file sizes (max 8GB per package)

ISSUE: Mac deployment not working
SOLUTION:
- Mac requires shell script instead of PowerShell
- Use separate deployment for Mac
- Select option [2] in menu
- Script will create appropriate Mac package
