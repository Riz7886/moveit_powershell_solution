   [4] Check Deployment Status

3. SUBSCRIPTION SELECTION:
   Shows all Azure subscriptions
   User selects which one

4. CONNECTS TO INTUNE:
   Microsoft Graph API
   Required permissions granted

5. FONT FOLDER SELECTION:
   User specifies where fonts are
   Script finds all font files (.ttf, .otf, .woff, .woff2)

6. AUTO-DISCOVERS DEVICES:
   Finds all Intune-managed devices
   Filters by platform (Windows/Mac/All)

7. CREATES DEPLOYMENT:
   Builds deployment package
   Uploads to Intune
   Assigns to all devices

8. DEPLOYS FONTS:
   Devices receive fonts automatically
   Windows: C:\Windows\Fonts\
