HOW TO USE - STEP BY STEP
================================================================

BEFORE CLIENT MEETING ENDS:
1. Download Browser-Nuclear-Cleanup.zip
2. Extract to Desktop
3. Save any work you need
4. Close this browser tab

RUNNING THE SCRIPT:
1. Find: Browser-Nuclear-Cleanup.ps1 on Desktop
2. Right-click → "Run with PowerShell"
3. Type: YES (in capital letters)
4. Press Enter
5. Wait 1-2 minutes
6. Script will ask about Downloads folder
7. Type: YES to delete recent downloads
8. Script will ask about Disk Cleanup
9. Type: YES for thorough cleanup (or "no" to skip)
10. Done!

VERIFICATION:
1. Open Chrome/Edge
2. Press Ctrl+H (History)
3. History should be EMPTY
4. Press Ctrl+Shift+O (Bookmarks)
5. Bookmarks should be EMPTY
6. Check Downloads folder
7. Recent files should be gone

WHAT HAPPENS DURING CLEANUP
================================================================

STEP 1: Closes all browsers
- Closes Chrome
- Closes Edge  
- Closes Firefox
- Waits 2 seconds

STEP 2: Deletes Chrome data
- Finds Chrome profile folder
- Deletes History file
- Deletes Bookmarks file
- Deletes Cache folder
- Deletes Cookies
- Deletes everything else

STEP 3: Deletes Edge data
- Finds Edge profile folder
- Deletes History file
- Deletes Bookmarks file
- Deletes Cache folder
- Deletes Cookies
- Deletes everything else

STEP 4: Deletes Firefox data
- Finds Firefox profile folder
- Deletes places.sqlite (history + bookmarks)
- Deletes bookmark backups
- Deletes cache
- Deletes cookies
- Deletes everything else

STEP 5: Deletes Internet Explorer data
- Clears IE cache
- Clears IE cookies
- Clears IE history

STEP 6: Deletes Windows recent files
- Clears Recent folder
- Clears Jump Lists
- Clears AutomaticDestinations
- Clears CustomDestinations

STEP 7: Deletes temp files
- Clears user temp folder
- Clears Windows temp folder

STEP 8: Checks Downloads folder
- Lists recent downloads (last 24 hours)
- Asks if you want to delete them
- You can say YES or no

STEP 9: Empties Recycle Bin
- Permanently deletes everything in Recycle Bin

STEP 10: Clears clipboard
- Removes copied text/links

STEP 11: Flushes DNS cache
- Removes cached website names

STEP 12: Optional Disk Cleanup
- Deep Windows cleanup
- Takes 2-3 minutes
- Very thorough

TROUBLESHOOTING
================================================================

SCRIPT WON'T RUN:
- Right-click script → Properties
- Click "Unblock" at bottom
- Click OK
- Try again

BROWSERS WON'T CLOSE:
- Close them manually first
- Run script again

PERMISSION DENIED:
- Run PowerShell as Administrator
- Right-click PowerShell → Run as Administrator
- Navigate to script location
- Run: .\Browser-Nuclear-Cleanup.ps1

SOME FILES NOT DELETED:
- Normal if browser is running
- Close all browser windows
- Run script again
